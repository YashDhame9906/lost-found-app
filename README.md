# Lost & Found Web Application:
A web application that helps users report and find lost items easily.

## Features
- Post lost or found items
- Upload item images
- View all items
- Simple and clean interface

## Tech Stack
- HTML, CSS
- PHP
- MySQL
- XAMPP

##  How to Run
1. Start XAMPP (Apache + MySQL)
2. Place project in:
   C:\xampp\htdocs\lostfound
3. Open:
   http://localhost/lostfound
